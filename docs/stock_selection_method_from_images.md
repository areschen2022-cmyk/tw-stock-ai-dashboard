# 圖片教材整理：如何加入台股 AI 選股方法

本文件整理使用者提供的 K 線、技術指標、基本面、估值與風險圖片，目的不是把所有訊號都直接加分，而是把它們拆成「題材」、「進場時機」、「品質」、「風險」四層檢查。

## 一、核心原則

1. 題材是股票進入觀察名單的原因，不等於買點。
2. 分級 S+、S、A 代表綜合強度，不代表可以無條件買進。
3. K 線型態只能作為輔助標籤，不能單獨產生 BUY_WATCH。
4. 財務與風險指標優先用來排雷，避免高負債、低流動性、自由現金流惡化的公司被題材分數拉高。
5. 所有新增規則都應先記錄標籤，再用 3 日、5 日、10 日績效驗證後，才調整分數權重。

## 三、技術層：K 線與指標只做「時機確認」

### 可作為偏多輔助標籤

這些型態只在「股價處於低位或整理後」、「成交量有確認」、「沒有重大風險警示」時才有參考價值：

| 型態 | 系統標籤 | 用法 |
|---|---|---|
| 啟明星 Morning Star | bottom_reversal | 見底反轉候選 |
| 陽包陰 Bullish Engulfing | bullish_engulfing | 多方吞噬，需量能確認 |
| 曙光初現 Piercing Line | piercing_line | 下跌後反彈訊號 |
| 錘子線 Hammer | hammer_bottom | 低位下影線承接 |
| 光頭光腳長紅 Marubozu | strong_body | 強勢表態，避免追高需看隔日 |

建議分數：先只加 `+2` 到 `+4` 的技術輔助分，且必須搭配「站上 MA20」、「量能延續」、「開盤不過度跳空」才可顯示為進場候選。

### 應作為風險警示

以下型態出現在高位或急漲後，應列為風險，不應加分：

| 型態 | 系統標籤 | 用法 |
|---|---|---|
| 黃昏星 Evening Star | top_reversal | 高位反轉警示 |
| 陰包陽 Bearish Engulfing | bearish_engulfing | 空方吞噬警示 |
| 烏雲蓋頂 Dark Cloud Cover | dark_cloud | 高位賣壓警示 |
| 上吊線 Hanging Man | hanging_man | 高位下影線，可能出貨 |
| 放量不漲 | volume_no_price | 資金倒貨或籌碼鬆動警示 |

建議處理：列入「危險名單」或降低操作結論，例如從「可追」降為「等拉回」或「只觀察」。

## 四、技術指標層：避免單一指標迷信

圖片提到 MA、MACD、RSI、KDJ、BOLL、ATR、OBV、成交量與換手率。系統應採共振邏輯：

| 類別 | 指標 | 加入方式 |
|---|---|---|
| 趨勢 | MA20/MA60、均線多頭排列 | 已有 MA20，可逐步補 MA60 與均線斜率 |
| 動能 | MACD、RSI、KDJ | 先做標籤，不直接大幅加分 |
| 量能 | 成交量、OBV、換手率 | 放量上漲加分；放量不漲列警示 |
| 波動 | ATR、BOLL | 用於進場距離與停損寬度，不直接判斷好壞 |

## 五、基本面層：挑好公司，不只挑熱門題材

圖片提到 EPS、ROE、毛利率、淨利率、營收成長、淨利成長、自由現金流。建議納入「品質分」：

| 指標 | 方向 | 用法 |
|---|---|---|
| EPS | 逐季或逐年成長 | 基本面加分 |
| ROE | 長期大於 15% 較佳 | 品質加分 |
| 毛利率 | 持平或上升 | 代表定價力 |
| 淨利率 | 持平或上升 | 代表管理效率 |
| 營收成長 | 持續成長 | 已有月營收，可補趨勢判讀 |
| 自由現金流 FCF | 正且穩定 | 風險排雷優先 |

## 六、估值與安全層：避免題材股變成追高股

圖片提到 PE、PB、PS、PEG、股息殖利率、資產負債率、流動比率、速動比率。建議先作風險標籤：

| 指標 | 警示條件 | 用法 |
|---|---|---|
| PE | 明顯高於同業且成長不足 | 估值過熱 |
| PB | 高於同業且 ROE 未跟上 | 淨值溢價過高 |
| PS | 高營收低獲利 | 需確認轉盈能力 |
| PEG | 遠大於 1 | 成長追不上估值 |
| 資產負債率 | 過高 | 財務壓力警示 |
| 流動比率/速動比率 | 過低 | 短期償債風險 |
| FCF | 長期為負 | 現金流風險 |

## 七、建議實作順序

1. 已完成：新增「石英元件/頻率控制」題材池。
2. 下一步：新增 K 線型態偵測器，但只輸出標籤，不直接改核心分數。
3. 再下一步：把「放量不漲」、「高位陰包陽」、「高位黃昏星」加入危險名單。
4. 後續：補自由現金流、負債率、流動比率、ROE、毛利率等基本面排雷欄位。
5. 最後：用 Signal Lab 回測每個標籤的 3 日、5 日、10 日勝率，再決定是否調整權重。

## 八、操作頁顯示建議

在今日監控頁中，未來可新增：

- `K線輔助`：啟明星、陽包陰、錘子線等，只顯示為輔助。
- `高位風險`：黃昏星、陰包陽、烏雲蓋頂、放量不漲。
- `財務排雷`：高負債、現金流弱、流動性不足。
- `操作結論`：可追、等拉回、只觀察、避開，優先讓使用者一眼看懂。

結論：圖片中的知識很適合加入系統，但應先當作「標籤與警示」，不要直接變成買進分數。真正可靠的做法是先記錄，再回測，再校準。
