# 進場、危險名單、題材品質規則

本系統的分級只代表訊號強度，不代表可以無條件買進。

## 今日操作檢查

- `S+ / S / A / B`：代表綜合強度。
- `entry_decision`：代表今日盤前操作狀態。
- `entry_checklist`：列出開盤後需要同時確認的條件。

常見狀態：

- `開盤確認`：標的強，但仍需確認站穩昨高、跳空未超限、前 5 分鐘量達標。
- `等拉回`：強度足夠，但不追高，等回測均線或近 3 日低點後守住。
- `量價不確認取消`：只觀察，若開盤沒有量價同時成立，就取消進場想法。
- `避免`：未達進場觀察門檻。

## 危險名單

危險名單用來抓「準備要跑」的訊號，包含：

- 外資或投信連續賣超。
- 法人賣超占成交量偏高。
- 融資增加但股價下跌。
- 跌破 MA5 / MA10 / MA20 或近 3 日低點。
- 爆量長黑，疑似資金撤退。
- 分數快速下降，或由買進觀察轉弱。

## 題材品質

題材熱度不只看新聞數，也標記命中品質：

- `高`：新聞含股票代號或公司名，和台股標的關聯較明確。
- `中`：多則產業關鍵字命中，但未明確點名公司。
- `低`：僅單一產業關鍵字命中，容易只是泛題材。

這些資訊僅供研究追蹤，不構成任何投資建議。
